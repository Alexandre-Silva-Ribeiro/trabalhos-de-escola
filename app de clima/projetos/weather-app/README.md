# Weather App

Simple and clean weather app using OpenWeather API.

## Description

This project lets users search weather by city and see location, temperature, weather description, icon, and local date/time.

## Features

- Search by city (with city + state support)
- Shows possible matches when city names are duplicated
- Displays:
  - city, state, country
  - temperature
  - weather description
  - weather icon
  - local date and time
- Dynamic background based on time, weather, and temperature

## Technologies

- HTML
- CSS
- JavaScript (Vanilla)
- OpenWeather Geocoding API
- OpenWeather Weather API

## Run locally

1. Open a terminal in this folder.
2. Run a simple local server:
   - `python -m http.server 3000`
3. Open:
   - `http://localhost:3000`
